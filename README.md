# StepMap (PWA, без бекенду)

StepMap — це PWA на чистому фронтенді (1 `index.html`), яка:
- зберігає налаштування в `localStorage`;
- рахує довжину кроку з вашого росту;
- показує, скільки лишилось кроків/метрів до цілі;
- будує пішохідний маршрут `WALKING` через Google Maps JavaScript API (Directions API) “туди і назад” на потрібну відстань.

Всі файли лежать в одній папці: `stepmap/`.

## 1) Google Maps API Key

В `index.html` є константа:

```js
const GOOGLE_MAPS_API_KEY = 'ВАШ_КЛЮЧ'
```

Щоб отримати (умовно безкоштовний) ключ:
1. Перейдіть у [Google Cloud Console](https://console.cloud.google.com/).
2. Створіть проєкт.
3. Увімкніть рахунок для платежів (часто це необхідно для використання Google Maps; навіть якщо є Free Tier/кредити).
4. Увімкніть API:
   - **Maps JavaScript API**
   - **Directions API**
5. Створіть **Credentials → API key**.
6. (Рекомендовано) Обмежте ключ:
   - HTTP referrers (домени вашого сайту).
7. Вставте ключ у `index.html`.

Якщо у вас буде помилка `Directions API ( ... )`, найчастіші причини:
- Directions API не увімкнений у цьому Google Cloud проєкті;
- неправильні HTTP referrer обмеження;
- ключ з неактивованим billing/квотою.

## 2) Запуск локально

Оскільки це PWA, браузеру потрібні `https` (або localhost) для Geolocation/Service Worker.

### Варіант A: простий статичний сервер
Наприклад, відкрийте папку через будь-який статичний сервер (Vite/serve/light-server тощо).

URL буде приблизно: `http://localhost:<port>/stepmap/`.

## 3) Деплой

Підходить для Vercel або GitHub Pages як статичний сайт.

### Vercel
1. Створіть проект з Git-препозиторію.
2. Вкажіть **Root directory** як `stepmap/`.
3. Build Command: залиште порожнім (без збірки).
4. Output Directory: залиште порожнім (статичний контент).

Після деплою переконайтесь, що в Google Cloud referrers доданий ваш домен (щоб Directions працював у браузері).

### GitHub Pages
1. Розмістіть вміст `stepmap/` у репозиторії так, щоб `index.html` був доступний як сторінка.
2. Увімкніть **GitHub Pages**.
3. Додайте URL вашого Pages-домену в **HTTP referrers** для ключа.

## 4) Додавання на iPhone

На iPhone в Safari:
1. Відкрийте сторінку StepMap.
2. Натисніть кнопку **Share**.
3. Оберіть **“Додати на екран”**.

## Нотатки про “туди і назад”

Додаток бере вашу поточну геолокацію, обчислює точку “виходу” наближеною відстанню (півдороги),
потім будує два walking-маршрути:
- Start → Out
- Out → Start

Після побудови Directions API рахується фактична довжина маршруту, яка відображається на карті у форматі:
`≈ {кроків} кроків ({метрів} м)`.
